# VGRAPH

Готовая версия приложения находится на ветке `build`.

## Скачать и запустить
1. Скачать ZIP репозитория с ветки `main`.
2. Извлечь файлы
3. Открыть `backend/dist` и запустить файл app.exe
4. Подождать пока загрузиться сервер
5. В браузере ввести http://localhost:5000/
