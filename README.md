# VGRAPH

Готовая версия приложения находится на ветке `build`.

## Скачать и запустить
1. Скачать ZIP репозитория с ветки `build`.
2. Извлечь файлы
3. В папке `build\dist` запустить app.exe
4. Дождаться пока запуститься сервер (консольное окно с сообщением * Running on http://127.0.0.1:5000)
5. В браузере выполнить: http://127.0.0.1:5000
