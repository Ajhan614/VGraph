# VGRAPH

Готовая версия приложения находится на ветке `build`.

## Скачать и запустить
1. Скачать ZIP репозитория с ветки `build`.
2. Извлечь файлы
3. Открыть папку `build` в Visual Studio Code
4. Установить Node.js и npm.
5. В терминале выполнить:
npm install -g http-server
http-server -p 3000 -c-1
